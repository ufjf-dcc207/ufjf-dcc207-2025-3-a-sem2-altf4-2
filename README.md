# TCG — Interface para criação de cartas colecionáveis

Uma interface para criação de cartas colecionáveis personalizadas. Este repositório contém o esqueleto do projeto (frontend) feito com Vite, React e TypeScript.

## Objetivo

Desenvolver a interface de um site onde usuários possam compor e visualizar cartas colecionáveis personalizadas.

## Tecnologias e dependências 

- Vite
- React
- TypeScript
- CSS
