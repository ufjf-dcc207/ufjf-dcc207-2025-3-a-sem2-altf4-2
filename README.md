# TCG — Interface para criação de cartas colecionáveis

Uma interface estática para criação de cartas colecionáveis personalizadas. Este repositório contém o esqueleto do projeto (frontend) feito com Vite, React e TypeScript. Para o primeiro seminário, o projeto está implementado de forma estática — sem backend nem persistência de dados — focando na apresentação visual e na experiência de criação de cartas.

## Objetivo

Desenvolver a interface de um site onde usuários possam compor e visualizar cartas colecionáveis personalizadas. Nesta primeira tarefa, o foco é na camada visual: layout, componentes e estilo, sem interatividade com base de dados.

## Tecnologias e dependências 

- Vite
- React
- TypeScript
- CSS
